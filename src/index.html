Scoring

Cards are scored as follows:

  • Each numeral card scores face value (Ace=1, Two=2, etc.)
  • Each Jack or Queen scores 10 points.
  • Pairing the same cards in a scoring column is scored as zero.
  • Each King scores zero points.
  • Joker scores as -2 points
  • Each 5 counts as -5
